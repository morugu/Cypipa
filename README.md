# Cypipa
Check your public IP address library written in Swift.

# Requirements
- iOS 8.0+
- Xcode 10.0+
- Swift 3.0+

# Installation

## Carthage
```
github "morugu/Cypipa"
```

```
carthage update
```

## CocoaPods
```
pod 'Cypipa'
```
```
pod install
```

```
Cypipa.getPublicIP { (ipAddress) in

}
```
