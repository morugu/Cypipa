# Cypipa
Check your public IP address library written in Swift.

# Requirements
- iOS 8.0+
- Xcode 10.0+
- Swift 3.0+

# Installation

## Carthage
Add your `Cartfile`
```
github "morugu/Cypipa"
```

Run following command
```
carthage update
```

```

}
```
